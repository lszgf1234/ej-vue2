| Prop | Type | Default | Description |
|---|---|---|---|
| `width` | Number | 1920 | 注入宽度document.documentElement.clientWidth |
| `height` | Number | 1080 | document.documentElement.clientHeight |
| `delay` | Number | 500 | 窗口去抖动，延迟时间 |
